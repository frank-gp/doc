# Para Spring Boot: usar Spring Initializr (opcionalmente desde CLI)

```sh
curl https://start.spring.io/starter.zip \
  -d dependencies=web \
  -d name=helloworld \
  -d packageName=com.frankgp \
  -o helloworld.zip

unzip helloworld.zip
cd helloworld
./mvnw spring-boot:run
```
