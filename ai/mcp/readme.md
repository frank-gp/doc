# Model Context Protocol (MCP) para Playwright

El Model Context Protocol (MCP) es un protocolo que permite a los desarrolladores de herramientas y servicios interactuar con modelos de lenguaje de manera estandarizada. En este caso, se utiliza para integrar Playwright, una herramienta de automatizaci√≥n de pruebas para aplicaciones web, con el protocolo MCP.
